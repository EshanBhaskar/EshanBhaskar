## 🚀 Featured Projects

### 📱 NSS IITP Official App

> A production-grade campus management platform with real-time chat, automated scheduling, and attendance tracking.

- **Stack:** Kotlin, Jetpack Compose, Firebase, Python, Node.js
- **Security:** Engineered hardware-bound QR attendance using Play Integrity API + ECDSA device binding.
- **Scale:** 185+ Kotlin files, 20+ Compose screens, custom TFV algorithm for teaching slot assignments.

### 🚖 CabShare

> Ride-sharing app for IIT Patna students — find cab partners to Patna Junction, Airport & Bihta Station.

- **Stack:** Kotlin, Jetpack Compose, Firebase, Microsoft Azure MSAL
- **Auth:** Secured via Microsoft MSAL, strictly restricted to `@iitp.ac.in` institutional emails.
- **Features:** Real-time ride matching, in-ride chat, automated past-ride cleanup using WorkManager.
