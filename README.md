## ⚡ Hey there! I'm Eshan

I'm a CS undergrad at **IIT Patna** who builds production Android apps — not just side projects.

- 🔭 **Currently:** Maintaining the [NSS IITP Official App](https://github.com/NSS-IITP-Official-App-Ecosystem/NSS-IITP-Official-App) — live on **Google Play Store** with real campus users
- 🔐 **Engineered** hardware-bound QR attendance using **Play Integrity API + ECDSA device binding** — unforgeable even on rooted devices
- 🧠 **Authored** a custom TFV constraint-satisfaction algorithm for automated teaching slot assignment
- 💬 Built real-time chat with Firebase Realtime DB, FCM push notifications, and Cloudinary media
- 🏆 Active competitive programmer on **LeetCode** & **Codeforces**
